# 📄 AI Resume Analyzer

![AI Resume Analyzer](screenshot.png)

---

## 🚀 Project Overview

AI Resume Analyzer is a Streamlit-based web application that helps job seekers evaluate their resumes against job descriptions.

The application extracts text from PDF resumes, identifies skills, compares them with job requirements, calculates an ATS (Applicant Tracking System) score, and provides personalized improvement suggestions.

It also generates a downloadable PDF report summarizing the analysis results.

---

## ✨ Features

- 📄 Resume PDF Text Extraction
- 🎯 Skill Detection from Resume
- 📝 Job Description Analysis
- 🤝 Skill Matching
- ❌ Missing Skill Identification
- 📊 ATS Score Calculation
- 📈 Skill Match Pie Chart Visualization
- 💡 Resume Improvement Suggestions
- 📥 Downloadable PDF Report

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/ManneSatwika/AI_Resume_Analyzer.git
```

### Move to Project Folder

```bash
cd AI_Resume_Analyzer
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
